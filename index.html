Course Overview

Frontend Developer

Master modern web development from scratch — HTML, CSS, JavaScript, and React.

About this Path

This comprehensive frontend course takes you from the basics of HTML and CSS all the way to building dynamic React applications. You'll learn how browsers work, how to write clean, semantic markup, how to style with modern CSS techniques, and how to bring pages to life with JavaScript. The final module covers React fundamentals and project deployment.

Learning Objectives

  • Build structured, accessible web pages using HTML5
  • Style responsive layouts with Flexbox and Grid
  • Write interactive JavaScript and manipulate the DOM
  • Improve website accessibility
  • Deploy static websites to the web
